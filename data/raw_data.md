# Raw Data

## Data Source
The raw data used in this analysis comes from publicly available Divvy (Cyclistic) bike-share trip data provided by Lyft Bikes and Scooters, LLC.

## Dataset Description
- Data consists of individual bike trip records
- Each record represents a single completed ride
- Data is provided as monthly CSV files

## Time Period Covered
- June 2022 through May 2023
- 12 consecutive months of data

## File Access
Due to file size limitations, raw CSV files are not stored in this repository.

The original datasets can be accessed here:
- https://divvy-tripdata.s3.amazonaws.com/index.html

## Key Raw Fields
- ride_id
- rideable_type
- started_at
- ended_at
- start_station_name
- end_station_name
- member_casual
